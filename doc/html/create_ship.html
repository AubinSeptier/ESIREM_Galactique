createShip-process.php

Cette fonction réalise le processus de création d'un vaisseau en utilisant les classes Research, Research_Type, Empire, Galaxy, Solar_System, Planet, Resource et Fleet. Elle récupère les données nécessaires depuis la superglobale $_GET et la superglobale $_SESSION. Elle effectue les vérifications nécessaires et crée un vaisseau avec les paramètres donnés.

La fonction effectue les étapes suivantes :

  • Vérifie si les paramètres requis ($_GET["ship"], $_GET["id_planet"]) sont définis.
  • Initialise les objets nécessaires (Research, Research_Type, Empire, Galaxy, Solar_System, Planet, Resource, Fleet).
  • Vérifie si le joueur a les recherches nécessaires pour construire le vaisseau.
  • Vérifie si le joueur a assez de ressources pour construire le vaisseau.
  • Crée un nouveau vaisseau avec les paramètres donnés.
  • Met à jour la flotte de la planète sélectionnée avec le nouveau vaisseau.
  • Retourne un message de succès avec les détails du vaisseau construit.
Exceptions
Exception_1Si les ressources nécessaires sont insuffisantes, renvoie un message d'erreur.
Exception_2Si les superglobales GET ne sont pas récupérées ou vides, renvoie un message d'erreur.
Remarques
Actuellement, les vaisseaux peuvent être construits sans avoir les bâtiments nécessaires.
Avertissement
Problème que nous n'avons pas réussi à résoudre : La méthode updateFleet() n'actualise pas la flotte de la planète sélectionnée. La méthode SQL fonctionne pourtant si on la teste de façon brut directement dans MariaDB.
